# 消融总表:每个预处理组件到底值不值

> 把实验 1–4 的结论汇成一张主表,回答项目总问题:
> **噪声 + 重叠条件下,哪些预处理真有用,哪些看起来高级却不划算?**

## ⚠️ 口径声明(读表前必看)

三个实验数据域不同,**绝对 CER 不可跨行直接比**,跨行只比"净效果方向":

| 域 | 数据 | 用途 |
|---|---|---|
| A | 单说话人 26 条 × 噪声(实验 1) | 测**降噪** |
| B | 自制重叠双人 11 对 × 噪声 × 重叠档(实验 2) | 测**分离 / 处理顺序** |
| C | 5 条代表性 ASR 错误(实验 3) | 测**LLM 纠错** |

CER 为字符错误率(越低越好);实验 2 用 content CER(15 个 cond×level 格的均值)。

## 主表:处理链路消融

| # | 在 baseline 上叠加 | 域 | 代表 CER(%) | 净效果 | 唯一值得的窄条件 |
|---|---|---|---|---|---|
| 0 | **baseline 直接 ASR** | A/B | A: W 26.5 / F 30.3(6 噪声均值)·B: 39.8 | 基准 | **永远先试这条** |
| 1 | + 降噪(FRCRN / 谱减) | A | W 26.5→47.9 / 43.0 ❌·F 30.3→36.1 / **27.3** | 多害少救 | 仅 **FunASR × white × 低 SNR**(0dB 68.4→**27.5**) |
| 2 | + 分离(MossFormer2) | B | L1 39.8 → **L3 68.2** ❌（spk 88.1） | 几乎全害 | 仅 **heavy 重叠**(clean/heavy 50.0→46.9) |
| 3 | + 降噪 → 分离(L4) | B | L3 68.2 → 67.4 | 无改善 | 无 |
| 4 | + 分离 → 降噪(L5) | B | L4 67.4 → 65.3 | 无稳定差异 | 无(L5 微优但不稳) |
| 5 | + LLM 纠错 | C | 幻觉案例 raw 90.0 → llm 41.0；通顺错词 0 变化 | 只会拒幻觉 | **整段是幻觉/无关内容**时(综艺名、字幕残留) |

**一句话**:消融每加一层,大多数条件下 CER 不降反升。L1 直接 ASR 在实验 2 的 15 个格子里 **13 个最优**;分离链路(L3-5)仅 heavy 重叠的 2 个格子胜出。

## 分域支撑数字

### 域 A · 降噪(实验 1,CER%,无处理 → FRCRN → 谱减)

| 条件 | whisper | funasr |
|---|---|---|
| clean | 4.9 → 9.3 → 14.7 | 10.1 → 8.5 → 13.0 |
| white_0dB | 41.9 → 49.9 → 61.7 | **68.4 → 27.5** → 40.1 |
| babble_0dB | 71.7 → 92.5 → 83.7 | 43.3 → **95.1** → 46.2 |

- 降噪只在 **FunASR × white × 低 SNR** 一格大幅救场;Whisper 任何场景都不受益。
- babble + 神经降噪 = 灾难(FunASR 43.3→95.1):FRCRN 分不清目标人声与背景人声,把目标语音消掉。

### 域 B · 分离 / 顺序(实验 2,content CER 15 格均值)

| 链路 | content CER | spk CER | 15 格最优数 |
|---|---|---|---|
| L1 直接 | **39.8** | — | **13** |
| L2 降噪→ASR | 52.5 | — | 0 |
| L3 分离→ASR | 68.2 | 88.1 | 2(均 heavy) |
| L4 降噪→分离 | 67.4 | 84.9 | 0 |
| L5 分离→降噪 | 65.3 | 85.2 | 0 |

- spk CER 始终 84–88%:MossFormer2 对 2–4s 短片段**根本没分开**,把混合复制到两路。
- L4 vs L5 差异小、无稳定赢家 → "先降噪更稳"假设不成立。

### 域 C · LLM 纠错(实验 3,content CER)

| 案例类型 | raw | 纯 LLM | 热词 |
|---|---|---|---|
| 术语错·通顺(好意) | 0.158 | 0.158 | 0.158 |
| 术语错·声学差远(玩具) | 0.40 | 0.40 | **1.0** |
| 近音错(说/多) | 0.077 | 0.077 | 0.077 |
| 综艺幻觉 | 2.286 | **0.571** | 0.571 |
| 字幕幻觉 | 1.579 | **0.842** | 0.842 |

- LLM 唯一可靠价值 = **识别并拒绝幻觉**(输出 `[无法识别]`,不脑补);声学信息已丢的通顺错词修不了;热词表无增益甚至帮倒忙。

## 反直觉发现汇总(项目核心 insight)

1. **两模型弱点相反**:Whisper 抗 white、怕 babble(诱发幻觉);FunASR 相反。0dB 处两条退化曲线交叉。
2. **"高级预处理"在短片段普遍负收益**:分离/降噪在 2–4s 辩论片段上几乎全程帮倒忙,只有 heavy 重叠勉强回本。
3. **处理顺序几乎不重要**:L4/L5 差异小,推翻"先降噪再分离更稳"的直觉。
4. **降噪改变 Whisper 错误模式**:幻觉 → 沉默(降噪后大量空转写)。
5. **VAD 有边界**(实验 4):压得住白噪/静音(100%→0%),压不住 babble(67%→67%)——人声型噪声骗过 VAD,正是 babble 0dB 最毒的根因。
6. **LLM 不是纠错器是过滤器**:能拒幻觉,修不了声学已丢失的通顺错词。

## 实践结论:推荐链路(规则版)

```
噪声 babble 且 SNR 低  → 直接 ASR(降噪会消人声;Whisper 易幻觉,可加 VAD 兜底)
噪声 white 且 SNR 低    → FunASR + 神经降噪(唯一稳定正收益组合)
重叠 no / light         → 直接 ASR(分离纯属画蛇添足)
重叠 heavy(长时抢话)   → 才考虑分离,且预期收益有限
输出疑似幻觉/无关内容    → LLM 过滤为 [无法识别],不要指望它修字词
```

**总结论**:在 2–4s 短辩论片段 + 噪声/重叠约束下,**最简单的直接 ASR 几乎总是性价比最高**;复杂前端只在极窄条件(FunASR×white×低SNR 的降噪、heavy 重叠的分离)才回本。这正面回答了项目总问题——大多数"高级预处理"在真实约束下不划算。
